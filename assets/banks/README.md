## Fuentes recomendadas para descargar los logos:

1. **Wikimedia Commons** - Muchos logos están disponibles en formato PNG/SVG
2. **SeekLogo.com** - Logos en formato PNG y vector
3. **LogosArchive.com** - Base de datos de logos
4. **Sitios oficiales de los bancos** - Para obtener los logos más actualizados

## Tamaño recomendado:
- Tamaño: 16x16 píxeles (o más grande, se escalará automáticamente)
- Formato: PNG con fondo transparente (preferible)
